# 분할 로켓

한 박을 1·2·3·4개로 고르게 나누는 경험을 통해 4분음표, 8분음표, 셋잇단음표, 16분음표를 배우는 모바일 리듬 게임입니다.

## 학습 흐름

듣기 → 연료칸을 리듬에 맞춰 채우기 → 기보 선택 → 타이밍 분석 → 새 혼합 패턴 응용 순서로 진행합니다. 연주 정확도, 기보 이해, 응용 점수는 결과 화면에서 분리해 보여 줍니다.

## 조작

- 모바일: 화면 하단 `연료 탭`
- 키보드: `Space` 또는 `A` `S` `D` `F`
- 설정: 소리, 카운트인, 진동, 시각 보조, 왼손 모드, 모션 강도, ±200ms 입력 보정

## 개발

```bash
npm install
npm run dev
npm run test:game
npm test
```

게임 타이밍은 `AudioContext.currentTime`을 기준으로 하며 Canvas 렌더링과 CSS 애니메이션은 판정에서 분리됩니다.
