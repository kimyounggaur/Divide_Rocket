## Runtime visuals and sound

- The stage, rocket, clouds, fuel rail, particles, icons, panels, and interaction effects are rendered from original project CSS and Canvas 2D code.
- All tones are synthesized at runtime with the Web Audio API.
- No third-party character art, commercial-game UI, music, sound recording, icon pack, animation library, or stock image is included.
